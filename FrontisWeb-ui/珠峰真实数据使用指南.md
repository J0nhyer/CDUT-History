# 珠峰1:1真实建模使用指南

## 🏔️ 三种实现方案

我已经创建了增强版组件 `AchievementPeakTrailRealistic.vue`，支持三种珠峰建模方式：

### 方案1：使用真实高度图（推荐）⭐

**优点**：基于NASA卫星数据，精度高，文件小
**缺点**：需要下载高度图

#### 步骤：

1. **获取珠峰高度图**
   
   访问：https://tangrams.github.io/heightmapper/
   
   - 在搜索框输入：`Mount Everest` 或坐标 `27.9881, 86.9250`
   - 调整地图到合适范围（建议Zoom 13-14，覆盖10-15km）
   - 点击右上角 **"Export"** 按钮
   - 下载PNG文件，重命名为 `everest-heightmap.png`

2. **放置文件**
   ```
   FrontisWeb-ui/FrontisWeb-ui/public/everest-heightmap.png
   ```

3. **使用组件**
   ```vue
   <AchievementPeakTrailRealistic 
     :personName="personData.name"
     :achievements="processedAchievements"
     heightmapPath="/everest-heightmap.png"
   />
   ```

---

### 方案2：使用3D模型文件

**优点**：最真实，专业制作
**缺点**：文件大，加载慢

#### 步骤：

1. **下载珠峰3D模型**

   **免费资源**：
   - Sketchfab: https://sketchfab.com/3d-models/mount-everest-90510edf11fd41fc8c886a87feab1c62
   - Free3D: https://free3d.com/3d-models/mount-everest
   
   **搜索关键词**：
   - "Mount Everest GLB"
   - "Himalaya 3D model"
   - "珠穆朗玛峰 GLTF"

   **格式要求**：GLB 或 GLTF（Three.js原生支持）

2. **转换格式（如果需要）**
   
   如果模型是OBJ、FBX等格式，使用在线工具转换：
   - https://products.aspose.app/3d/conversion
   - 上传模型 → 选择输出格式为GLB → 下载

3. **放置文件**
   ```
   FrontisWeb-ui/FrontisWeb-ui/public/models/everest.glb
   ```

4. **使用组件**
   ```vue
   <AchievementPeakTrailRealistic 
     :personName="personData.name"
     :achievements="processedAchievements"
     modelPath="/models/everest.glb"
   />
   ```

## 🚀 快速开始（推荐方案1）

### 第1步：获取高度图

1. 打开浏览器访问：https://tangrams.github.io/heightmapper/
2. 搜索框输入：`27.9881, 86.9250`（珠峰坐标）
3. 调整视图：
   - Zoom级别：13-14
   - 确保珠峰位于中心
4. 点击 **Export** → 下载PNG

### 第2步：保存文件

将下载的PNG文件重命名为 `everest-heightmap.png`，放到：
```
FrontisWeb-ui/
  └── FrontisWeb-ui/
      └── public/
          └── everest-heightmap.png  ← 这里
```

### 第3步：修改组件引用

编辑 `PersonDetailAcademic.vue`：

```vue
<script>
// 导入真实版本
import AchievementPeakTrail from './AchievementPeakTrailRealistic.vue'
</script>

<template>
  <!-- 在荣誉成就部分使用 -->
  <AchievementPeakTrail 
    :personName="personData.name"
    :achievements="processedAchievements"
    heightmapPath="/everest-heightmap.png"
  />
</template>
```

### 第4步：测试

```bash
npm run dev
```

访问人物详情页 → 点击"荣誉成就"标签 → 查看真实珠峰！

## 🛠️ 高级定制

### 调整山峰高度

编辑 `AchievementPeakTrailRealistic.vue`，找到：

```javascript
displacementScale: 100  // 修改这个数值
```

- 增大 = 山峰更高
- 减小 = 山峰更平缓

### 调整材质颜色

```javascript
color: 0x8B7355  // 岩石颜色（十六进制）
```

常用颜色：
- `0x8B7355` - 棕褐色（岩石）
- `0xFFFFFF` - 纯白色（雪山）
- `0x696969` - 灰色（花岗岩）

### 调整攀登路线

```javascript
const angle = t * Math.PI * 3.5  // 螺旋圈数
const radius = 80 * (1 - t * 0.85) + 10  // 螺旋半径
```

## ❓ 常见问题

### Q: 高度图显示不出来？
**A**: 检查：
1. 文件路径是否正确（在public文件夹内）
2. 文件名是否匹配
3. 浏览器控制台是否有404错误

### Q: 加载很慢？
**A**: 
- 如果用高度图：尝试降低PNG分辨率到1024×1024
- 如果用3D模型：使用GLB格式而不是GLTF+bin

### Q: 想用雪山效果？
**A**: 修改材质颜色为白色：
```javascript
color: 0xFFFFFF,
roughness: 0.6,  // 雪的粗糙度
```

### Q: 性能问题？
**A**: 降低网格分辨率：
```javascript
// 从 512x512 改为 256x256
new THREE.PlaneGeometry(300, 300, 256, 256)
```
